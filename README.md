# Технологический стек

- PHP 8.1+
- MySQL 8.0
- Smarty 4.3
- PDO
- Faker
- Docker

## Реализованные страницы

1. **Главная страница** (`/`)
   - Отображение категорий с активными статьями
   - 3 последних поста по дате публикации для каждой категории
   - Кнопка "Все статьи" для каждой категории

2. **Страница категории** (`/category/{slug}`)
   - Название и описание категории
   - Список всех статей категории
   - Сортировка по дате публикации или количеству просмотров
   - Пагинация (9 статей на страницу)

3. **Страница статьи** (`/article/{slug}`)
   - Полная информация о статье
   - Изображение, название, описание, текст
   - Категории статьи
   - Количество просмотров
   - Блок из 3 похожих статей

## Установка и запуск

### Шаг 1: Запуск Docker контейнеров

```bash
docker-compose up -d
```

Это запустит:
- PHP 8.1 (порт 8080)
- MySQL (порт 3306)
- phpMyAdmin (порт 8081)

### Шаг 2: Установка зависимостей

```bash
docker exec -it blog_php composer install
```

### Шаг 3: Создание структуры БД

```bash
docker exec -it blog_php php seed.php
```

Этот скрипт:
- Создаст таблицы в базе данных
- Заполнит БД тестовыми данными (6 категорий, 50 статей)

### Шаг 4: Визуал

Откройте в браузере:
- **Блог**: http://localhost:8080
- **phpMyAdmin**: http://localhost:8081
